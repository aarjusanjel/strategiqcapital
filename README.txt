StrategiQ Capital Advisory — Website Files
===========================================

HOW TO VIEW
Open index.html in any browser. Click around — all 5 pages
(Home, About & Team, Services, How We Work, Contact) link to each other.

HOW TO ADD TEAM PHOTOS LATER
Inside the "assets/team" folder, add a photo for each person using
EXACTLY these filenames (any of .jpg works once renamed to match):
  - arju.jpg     (Arju Sanjel)
  - satish.jpg   (Satish Khadka)
  - ashish.jpg   (Ashish Sanjel)
  - sanjay.jpg   (Sanjay Gautam)
  - saroj.jpg    (Saroj Basnet)
The moment a correctly-named file is dropped into assets/team/, it
automatically replaces the grey "photo pending" placeholder on the
About & Team page — no code changes needed.

HOW TO PUBLISH IT
These are static files, so any basic web host works: Netlify, Vercel,
GitHub Pages, or your own hosting provider. Just upload the whole
"strategiq-website" folder (keeping the assets/ subfolder structure intact).
